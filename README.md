# Password Manager 

A simple password manager built using **Python** and **Tkinter**.  
The application can generate strong random passwords, copy them to the clipboard, and save login details locally.

## About the Project
This project was created to practise GUI development and event-driven programming in Python. It demonstrates how user input, file handling, and random password generation can be combined into a practical desktop application.

## Features
- Generate strong random passwords
- Automatically copy generated passwords to the clipboard
- Save website, email/username, and password details to a local file
- Input validation with error and confirmation pop-ups
- Simple and clean Tkinter user interface

## Technologies Used
- Python
- Tkinter (GUI)
- `random` module
- `pyperclip` for clipboard copying
